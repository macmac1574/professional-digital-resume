# Professional Digital Resume

This project is a personal digital resume created for the
Web Systems and Technologies course. It demonstrates the use
of clean typography, spacing, and layout principles using
Tailwind CSS.

## Project Features

- About Me, Technical Skills, and Contact sections
- Clean and modern typography
- Proper use of padding, margin, borders, and shadows
- Responsive layout using Tailwind CSS utilities

## Technologies Used

- HTML5
- Tailwind CSS
- Git & GitHub

## How to Open

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

## Brief Explanation of Why I Chose this project

" I chose the Professional Digital Resume project because
it allowed me to demonstrate my understanding of web
structure, typography, and the CSS box model using Tailwind
CSS. This project focuses on creating a clean and Professional
layout while properly applying spacing, borders, and shadows.
It also reflects a real-world use case where a personal website
can be used to professionally present skills, education,
experience, and contact information. "
